# BetaX Technologies Website

Official website for BetaX Technologies - Nigerian-based technology and innovation company focused on smart agriculture, IoT solutions, and sustainable energy systems.

## Features

- **Admin Dashboard**: Netlify CMS-powered content management at `/admin`
- **Dynamic Content**: All content loaded from `_data` directory
- **Products Management**: Add, edit, reorder products
- **Team Management**: Manage team members
- **News/Blog**: Create and publish news with text, images, or videos
- **Contact Info**: Centrally managed contact information

## Tech Stack

- HTML, CSS, JavaScript
- Netlify CMS for content management
- Netlify Identity for authentication
- Netlify Hosting

## Local Development

1. Install Python (if not already installed)
2. Run local server:
   ```bash
   python -m http.server 8000
   ```
3. Open browser to `http://localhost:8000`

## Admin Access

- Admin Dashboard: `yoursite.com/admin`
- Login with email/password via Netlify Identity

## Deployment

Deployed automatically via Netlify when pushing to GitHub.
